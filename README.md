# News
Demo app which displays collection of news fetched from https://newsapi.org.

| User Interface     | SwiftUI & Combine |
| ------------------ | ----------------- |
| **Network**        | `URLSession`      |
| **Design Pattern** | **MVVM**          |
| **Compatibility**  | **iOS 14.0+**     |
| **Swift**          | **5**             |
